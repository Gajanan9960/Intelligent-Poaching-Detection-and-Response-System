# DESIGN.md - Poaching Detection System

## 1. Description
Intelligent Poaching Detection and Response System UI. Must feel like a modern, award-winning monitoring dashboard.

## 2. Core Themes
- **Wildlife Conservation:** Deep greens, forest aesthetics, nature-inspired minimalism.
- **High-Tech Surveillance:** Clean typography, glassmorphism, dynamic data visualization elements.

## 3. Colors
- **Primary:** `text-emerald-400`, `bg-emerald-600`
- **Secondary:** `text-green-300`, `bg-green-800`
- **Background:** Very dark forest green/black `#06140b` fading to `#0a2916`
- **Alerts:** Vibrant red for suspicious activity `#ef4444`
- **Surface:** Translucent glassmorphism (e.g., `bg-white/10 backdrop-blur-md border border-white/20`)

## 4. Typography
- **Headings:** Inter or Outfit, clean sans-serif.
- **Data/Dashboards:** Monospace for confidence scores and timestamps.

## 5. UI Elements
- **Buttons:** Smooth hover states, glowing gradients.
- **Cards:** Glassmorphic containers with soft internal shadows.

## 6. Design System Notes for Stitch Generation
The UI must use real TailwindCSS classes with a dark forest theme: background `#06140b`. Use `bg-white/5 backdrop-blur-lg` for panels. Highlight suspicious AI detections (poacher, weapon) with bold red `#ef4444` borders or text, and animals/rangers in neutral/green.`
